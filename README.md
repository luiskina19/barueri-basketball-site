 Barueri Basketball Website
 Sobre o Projeto

Site institucional desenvolvido para apresentar informações sobre a equipe de basquete Barueri, incluindo calendário de jogos, localização, parceiros e contatos.

Objetivo

Aplicar e consolidar conhecimentos em desenvolvimento web, utilizando HTML, CSS e JavaScript na construção de uma aplicação organizada e responsiva.

 Tecnologias Utilizadas

HTML5

CSS3

JavaScript

Funcionalidades

Apresentação do time

Exibição de jogos

Informações de localização

Área de colaboradores

Seção de contato
